# User Subscriptions
##v1.0
### Requirements:
* PHP 7.3
* MySQL 10
### Installation Instructions:
* Open a terminal
* Run the commands below
```
// clone project:
git clone https://github.com/eze2411/user-subscriptions.git
```
```
// install dependencies:
cd user-subscriptions
composer install
npm install
```
### Start Application:
```
php artisan serve
```
